## 새로운 시작

기존에 Hexo로 운영하던 블로그를 완전히 새로 구축했습니다.

### 기술 스택

이번 블로그는 다음 기술 스택으로 만들었습니다:

- **Astro** - 정적 사이트 생성기
- **React** - 인터랙티브 컴포넌트
- **Tailwind CSS 4** - 스타일링
- **study-ui-lib** - 자체 제작 UI 컴포넌트 라이브러리

### 앞으로의 계획

시스템 프로그래밍, 네트워크, 보안 등 학습한 내용을 정리하고 공유할 예정입니다.

주로 다음 주제를 다룰 계획입니다:

- Linux Kernel / Network 프로그래밍
- C++ / Rust 시스템 프로그래밍
- 웹 보안 기술 (TLS, eBPF)
- 아키텍처 설계 및 성능 최적화
