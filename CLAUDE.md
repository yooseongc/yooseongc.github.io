# yooseongc.github.io

개인 프로필, 기술 블로그, 개인 프로젝트 소개를 위한 정적 github.io 블로그
기존에 작성한 것(old/ 로 옮겨놓음)은 갈아엎습니다.

## 개발 기술

 - astro + react.js를 이용한 정적 사이트 제작
 - ../study-ui-lib 을 사용한 컴포넌트 UI 활용. 해당 라이브러리의 스타일 가이드를 준수.
 - 컨텐츠는 markdown으로 관리
 - 배포는 github action > github pages

## 기술 블로그 컨텐츠

- contents/ 를 참고하여 작성
- 깔끔한 기술블로그 풍의 디자인. 필요하다면 직접 내용이나 이미지를 추가하는 것을 허용

## Rules

- README.md 를 적절하게 만들기
- docs/PROCESS.md에 항상 작업 내용을 기록
- docs/ISSUES.md에 문제 사항 및 해결 방법을 기록
- docs/RULES.md에 작업 간 지켜야 할 사항을 기록
- skill을 적절히 만들어 활용할 것
- 코드는 항상 간결하고 잘 구분되도록 할 것
- 페이지 간 일관성을 항상 유지할 것
